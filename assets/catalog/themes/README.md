# Full Theme Layers

These are the big desktop-level theme choices users usually compare.

## Major Layers

| Layer | What it affects |
|-------|-----------------|
| KDE Colorscheme | Window chrome, menus, titlebars |
| Kvantum | Qt widgets: buttons, inputs, dropdowns, checkboxes |
| Plasma Theme | Panel/taskbar SVG look |
| Cursor Theme | Mouse cursor |
| GTK Theme | Non-Qt app chrome |
| Wallpaper | Overall visual anchor |
| Icon Theme | App/file icons |
| Splash / Login | Boot/login branding |

## Recommendation Flow

1. Pick color palette
2. Pick Qt widget style (Kvantum)
3. Pick panel style (Plasma theme)
4. Pick cursor theme
5. Pick GTK fallback
6. Pick wallpaper
7. Optional: icon theme and splash screen

## Current ricer support

- Supported: KDE colorscheme, Kvantum, Plasma theme, cursor, wallpaper
- Documented / planned: GTK theme
- Not yet implemented: icon theme, splash, SDDM theme

## Highest visual impact

1. Wallpaper
2. Kvantum
3. Plasma theme / panel
4. Terminal colors
5. Cursor
