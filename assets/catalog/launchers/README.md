# Launcher Options

Launchers control app search, window switching, and quick commands.

## Common Picks

| Launcher | Best for | Notes |
|----------|----------|-------|
| rofi | Most popular | Themeable, scriptable, widely used |
| wofi | Wayland equivalent | Lightweight |
| dmenu | Minimal classic | Extremely simple |
| Albert | GUI launcher | Plugin ecosystem |

## Current ricer support

- Supported: rofi
- Not yet implemented: wofi, dmenu

## What users usually customize

- Window border and radius
- Prompt style
- Search result highlight
- Font and spacing
- Transparency / blur

## Recommendation Flow

- Want the standard ricer choice? → rofi
- Want Wayland-native? → wofi
- Want ultra-minimal? → dmenu
