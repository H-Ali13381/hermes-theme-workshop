## ScreenProjection : [WindowsetProjection ](/docs/v0.3.0/types/Quickshell.WindowManager/WindowsetProjection)
uncreatable

`import Quickshell.WindowManager `
A ScreenProjection is a special type of [WindowsetProjection ](/docs/v0.3.0/types/Quickshell.WindowManager/WindowsetProjection)which aggregates all windowsets across all projections covering a specific screen.

When used with [Windowset.setProjection() ](/docs/v0.3.0/types/Quickshell.WindowManager/Windowset#setProjection), an arbitrary projection on the screen will be picked. Usually there is only one.

Use [WindowManager.screenProjection() ](/docs/v0.3.0/types/Quickshell.WindowManager/WindowManager#screenProjection)to get a ScreenProjection for a given screen.
