## WlrLayer : WlrLayer

`import Quickshell.Wayland `
See [WlrLayershell.layer ](/docs/v0.3.0/types/Quickshell.Wayland/WlrLayershell#layer).

## Variants

* Background

Below bottom

* Bottom

Above background, usually below windows

* Top

Commonly used for panels, app launchers, and docks. Usually renders over normal windows and below fullscreen windows.

* Overlay

Usually renders over fullscreen windows
