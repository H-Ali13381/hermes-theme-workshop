* activeWorkspace : [I3Workspace ](/docs/v0.3.0/types/Quickshell.I3/I3Workspace)

The currently active workspace on this monitor, May be null.

* name : [string ](https://doc.qt.io/qt-6/qml-string.html)
readonly

The name of this monitor

* power : [bool ](https://doc.qt.io/qt-6/qml-bool.html)
readonly

Whether this monitor is turned on or not

* height : [int ](https://doc.qt.io/qt-6/qml-int.html)
readonly

The height in pixels of this monitor

* focused : [bool ](https://doc.qt.io/qt-6/qml-bool.html)
readonly

Whether this monitor is currently in focus

* id : [int ](https://doc.qt.io/qt-6/qml-int.html)
readonly

The ID of this monitor

* lastIpcObject : [unknown ](#unknown)
readonly

Last JSON returned for this monitor, as a JavaScript object.

* y : [int ](https://doc.qt.io/qt-6/qml-int.html)
readonly

The Y coordinate of this monitor inside the monitor layout

* width : [int ](https://doc.qt.io/qt-6/qml-int.html)
readonly

The width in pixels of this monitor

* scale : [real ](https://doc.qt.io/qt-6/qml-real.html)

The scaling factor of this monitor, 1 means it runs at native resolution
