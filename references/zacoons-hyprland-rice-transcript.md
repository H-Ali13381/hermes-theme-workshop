## Notable details for the ricing skill

- **Compositor:** Hyprland (Wayland), with custom C++ plugins.
- **Widget toolkit:** QuickShell (QML/Qt) — chosen over Eww.
- **Custom plugins:**
  - Extended IPC `closewindow` event to include window class (so close-sound only fires for terminals).
  - **Image borders plugin** — 9-slice tiling of an image around windows; required a small patch to the Hyprland plugin API. Per-window opt-out supported.
- **Audio:** open/close SFX from freesound.org, gated by IPC events.
- **Theming:** dark/light theme switching; pivoted from hand-drawn Skyrim-style art to pixel art mid-competition.
- **Other widgets:** parallax wallpaper (foreground/background cutout), battery indicator, launcher icon, notification popups, media dropdown, screenshot tool with rope physics (Qt Quick control, valet integration), lockscreen (added post-submission).
- **Dropped ideas:** Godot as desktop renderer, Lord-of-the-Rings "magic basin" launcher, vignette / bloom / chromatic aberration post-processing.
