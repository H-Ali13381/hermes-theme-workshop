# Bar Options

Status bars are one of the highest-impact ricing elements.

## Common Picks

| Bar | Best for | Notes |
|-----|----------|-------|
| KDE Panel | Native Plasma integration | Widgets, taskbar, system tray |
| waybar | Wayland ricing | CSS-stylable, modular |
| polybar | X11 / classic ricing | Highly customizable |
| i3blocks | i3 minimal setups | Simple block-based status |

## Recommendation Flow

- Using KDE and want native widgets? → KDE Panel
- Using Wayland compositors? → waybar
- Using X11 tiling WM? → polybar

## Current ricer support

- Supported: KDE panel theme (via Plasma theme), waybar
- Not yet implemented: polybar

## What users usually customize

- Background / opacity
- Rounded corners
- Height / padding
- Task button styling
- Workspace indicators
- Module separators
- Clock, network, battery, volume appearance
