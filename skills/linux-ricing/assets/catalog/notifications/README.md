# Notification Options

Notification daemons shape popups, urgency colors, borders, and placement.

## Common Picks

| Daemon | Best for | Notes |
|--------|----------|-------|
| dunst | Most popular / X11 / mixed setups | Simple, themeable |
| mako | Wayland-native minimal | Good for sway/hyprland |
| swaync | Notification center | Richer UI |

## Current ricer support

- Supported: dunst
- Not yet implemented: mako, swaync

## What users usually customize

- Border color
- Background color
- Text color
- Urgency levels
- Transparency
- Corner radius
- Screen position

## Recommendation Flow

- Want standard and simple? → dunst
- Want Wayland-native? → mako
- Want a richer notification center? → swaync
